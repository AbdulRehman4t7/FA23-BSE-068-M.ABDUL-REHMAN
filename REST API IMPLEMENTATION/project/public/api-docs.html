🚀 REST API Documentation

Complete API Reference for Users, Books, and Items Management

📋 Base URL

http://localhost:3000/api/v1

🔐 Authentication

This API uses JWT (JSON Web Token) for authentication. Protected endpoints require a Bearer token in the Authorization header.

Authorization: Bearer YOUR_JWT_TOKEN_HERE
Note: Get your token by logging in through the /users/login endpoint.

👥 Users API

POST Register User

/users

Create a new user account

{
  "name": "Ali Ahmed",
  "email": "ali@example.com",
  "password": "securepass123",
  "department": "Computer Science"
}
201 Created 400 Bad Request 409 Conflict

POST Login

/users/login

Authenticate and receive JWT token

{
  "email": "ali@example.com",
  "password": "securepass123"
}
200 OK 401 Unauthorized

GET Get All Users

/users

Retrieve list of all users

200 OK

GET Get User by ID

/users/:id

Retrieve a specific user

200 OK 404 Not Found

PUT Update User 🔒 Auth Required

/users/:id

Update user profile (own profile only)

{
  "name": "Ali Ahmed Updated",
  "department": "Software Engineering"
}
200 OK 403 Forbidden 404 Not Found

DELETE Delete User 🔒 Auth Required

/users/:id

Delete user account (own account only)

204 No Content 403 Forbidden 404 Not Found

📚 Books API

GET Get All Books

/books?page=1&pageSize=10&author=Martin

Retrieve list of books with pagination and filtering

200 OK

POST Add Book 🔒 Auth Required

/books

Add a new book to the collection

{
  "title": "Clean Code",
  "author": "Robert C. Martin",
  "price": 500,
  "isbn": "978-0132350884"
}
201 Created 400 Bad Request 401 Unauthorized

GET Get Book by ID

/books/:id

Retrieve a specific book

200 OK 404 Not Found

PUT Update Book 🔒 Auth Required

/books/:id

Update book details

{
  "price": 550,
  "title": "Clean Code - Updated Edition"
}
200 OK 404 Not Found 401 Unauthorized

DELETE Delete Book 🔒 Auth Required

/books/:id

Remove a book from the collection

204 No Content 404 Not Found 401 Unauthorized

🔒 Security Features

  • ✅ CORS - Cross-Origin Resource Sharing configured
  • ✅ Helmet - Security headers (CSP, HSTS, etc.)
  • ✅ Rate Limiting - 500 requests per 15 minutes
  • ✅ JWT Authentication - Token-based auth
  • ✅ Input Validation - Express-validator
  • ✅ XSS Protection - Input sanitization
  • ✅ Error Handling - Centralized error responses
